Para executar, siga os passos abaixo:
1. copie a pasta "scripts" para dentro do projeto a ser analisado
2. Na linha de comando, entre na pasta scripts
3. Ainda na linha de comando, execute o comando: python extrai_expertise.py <cv>
<cv> corresponde ao sistema de controle de versões utilizado no projeto, são três
as opções possíveis: hg, svn ou git
